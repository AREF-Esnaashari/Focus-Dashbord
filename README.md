## 📥 Installation

Clone the repository:

```bash
git clone https://github.com/AREF-Esnaashari/App-Movies.git
```

Move into the project directory:

```bash
cd App-Movies
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Open your browser and visit the local address displayed in the terminal (usually `http://localhost:5173`).
